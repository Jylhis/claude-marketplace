# productivity

General productivity and workflow skills for session management, logging, and daily workflows.

## Install

```
/install productivity@jylhis-plugins
```

## Skills

| Skill | Description |
|-------|-------------|
| session-log | Summarize the current conversation session and append results to the weekly agent-log |
